Proyecto PENSER

Reconstrucción del Índice de Impacto de Egresados (IIE)

Equipo de Análisis de Datos y Estadísticas

Fecha de Presentación

Problema a Resolver: Inconsistencias en el IIE

  • Discrepancia de Datos: Se detectaron inconsistencias significativas entre los documentos de referencia y las bases de datos proporcionadas.
  • Falta de Trazabilidad: El proceso de cálculo anterior carece de la documentación y trazabilidad claras para validar los resultados.
  • Necesidad de Confianza: Urge reconstruir el Índice de Impacto de Egresados (IIE) para garantizar su precisión, confiabilidad y utilidad en la toma de decisiones.
Enfatizar que el problema no es solo un error, sino una falta de confianza en la métrica actual.

Objetivo del Proyecto

  1. Validar y Reconciliar: Analizar las fuentes de datos primarias para identificar y documentar las inconsistencias encontradas.
  2. Desarrollar una Metodología Sólida: Proponer y aplicar un proceso estadístico riguroso para la nueva construcción del IIE.
  3. Generar el IIE Reconstruido: Obtener un Índice de Impacto de Egresados (IIE) preciso, reproducible y totalmente documentado.
  4. Implementar Trazabilidad: Dejar una estructura de código y documentación que permita auditar y replicar el cálculo en el futuro.

Metodología: Enfoque PENSER

Fase 1: Auditoría y Limpieza de Datos

  • **Identificación de Fuentes:** Mapeo de bases de datos de egresados, resultados y parámetros del IIE.
  • **Detección de Inconsistencias:** Aplicación de reglas de validación para señalar discrepancias (documento vs. dato).
  • **Estandarización:** Normalización y limpieza de variables clave.

Fase 2: Modelado y Recálculo

  • **Definición de Variables:** Confirmación de las variables que componen el IIE.
  • **Recálculo del Peso:** Aplicación de la fórmula matemática o estadística definida para obtener el nuevo IIE.
  • **Herramientas:** Uso de R/Python para garantizar reproducibilidad (código abierto).
